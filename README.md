# 八字命盘

根据王德峰教授推荐的三本书《穷通宝鉴》《三命通会》《滴天髓》三部经典为算法基础的八字排盘工具。纯 Python 标准库，零依赖，一键启动 Web 界面。

## 快速开始

```bash
git clone git@github.com:Xv-nosay/bazi.git
cd bazi
python3 server.py
```

浏览器打开 `http://localhost:8768`，输入公历年月日时和性别即可排盘。

## 功能

- **四柱排盘** — 年/月/日/时柱干支、藏干、十神、纳音
- **用神调候** — 《穷通宝鉴》120 条规则自动匹配月令用神
- **格局判定** — 《三命通会》正官/七杀/财/印/食伤/建禄/阳刃/杂气八格
- **神煞系统** — 天乙贵人、文昌、桃花、驿马、羊刃、华盖、将星、空亡
- **旺衰四维** — 《滴天髓》得令/得地/得生/得助量化评分
- **刑冲合害** — 天干五合、地支六合三合半合、六冲、三刑
- **大运流年** — 顺逆排大运、起运岁数、当前大运与流年自动推算
- **综合解读** — 命局总览/性格/事业/财运/感情/健康/未来发展七维分析

## 依赖

无。纯 Python 标准库，不需要 `pip install` 任何东西。
